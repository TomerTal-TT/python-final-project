# python-final-project
Secure Chat System (Python)

Overview:

This project is a client-server chat application built in Python using sockets and threading.

The system allows up to 5 users to connect simultaneously, with authentication based on username and password.

Features
	•	Multi-user support (up to 5 clients)
	•	User authentication (login system)
	•	Real-time messaging
	•	Thread-based client handling

Technologies Used
	•	Python
	•	Socket programming
	•	Threading

How to Run

Server:
python exercise_server.py
Client:
python exercise_client.py


What I Learned
	•	Building client-server architecture
	•	Handling multiple users with threading
	•	Implementing basic authentication
	•	Working with network sockets
